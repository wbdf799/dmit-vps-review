## 几个你可能有的顾虑

**超量流量会断网吗？**

T1 系列超量后不断网，直接限速到 50-100Mbps 继续跑，月初自动重置。Pro 系列部分套餐也有类似机制。不用担心月底流量报警被迫停机。

**IP 被墙怎么办？**

Pro 和 Eyeball 系列，IP 被墙后可以免费更换，每 15 天一次。其他情况换 IP 收 $5/次。这个政策在同类商家里算透明——有规则，不是"联系客服看心情"。

**三天内不想用了能退吗？**

3 天内、使用流量不超过 30GB，可以全额退款（扣除支付网关手续费）。30 天内按剩余服务时间比例退剩余金额。建议先买最小套餐跑一个月摸摸底，再决定要不要升配。

**热门套餐真的会缺货吗？**

真的会。洛杉矶 Pro 的部分套餐经常处于缺货状态，香港和东京也有不少款常态无货。看到合适的库存要快，官网实时标注"In Stock"状态。
